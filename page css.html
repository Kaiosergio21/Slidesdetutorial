css

O que é ?

CSS (Cascading Style Sheet, ou Folha de Estilo em Cascatas), é necessário dizer que sua aplicação é utilizada na estilização de componentes escritos em linguagens de marcação.

Para que serve

Uma das grandes vantagens do CSS é a possibilidade de personalização praticamente inacabável, com diversas possibilidades de estilização em uma página HTML.
